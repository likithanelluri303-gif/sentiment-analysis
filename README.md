# Sentiment Analysis Tool

NLP-powered sentiment analysis application that processes customer reviews and provides sentiment scores.

## Features
- Text classification
- Sentiment scoring
- Customer review analysis
- Natural language processing

## Tech Stack
- Python
- NLP
- NLTK
- TextBlob
- Flask

## Getting Started

```bash
git clone https://github.com/likithanelluri303-gif/sentiment-analysis.git
cd sentiment-analysis
pip install -r requirements.txt
python app.py
```
